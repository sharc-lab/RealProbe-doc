Welcome to RealProbe's documentation!
===================================

**RealProbe** is an FPGA profiling tool working with Vitis HLS to probe the on-board performance of HLS designs.

Check out the :doc:`usage` section for further information, including
how to :ref:`installation` the project.

.. note::

   This project is under active development.
